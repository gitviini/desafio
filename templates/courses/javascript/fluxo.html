FLUXO DE DECISÃO

Nesse desafio nos deparamos com a parte que as decisões do usuário afetam o rumo dos acontecimentos.

Para isso vamos utilizar de algumas features (funções), como: for, while, if e else.

  • for : laço de repetição para cada elemento de algum conjunto
  • while : laço de repetição que será executado enquanto algo for verdadeiro
  • if : condição que faz algo caso sejá verdadeiro
  • else : condição que faz algo caso sejá falso

O desafio é criar um jogo que o usuário consiga escolher entre:

  • Front-end ou Back-end
  • Se Front
    • React ou Vue
  • se Back
    • C# ou Java
  • Especializar na área ou Fullstack
  • Quais tecnologias gostaria de se especializar (loop)
